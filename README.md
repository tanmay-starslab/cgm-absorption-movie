# CGM Absorption-Spectroscopy Movie — TNG50 sub 488530

Publication-grade animation for the Edinburgh talk: a background-QSO sightline probing the
multiphase CGM of TNG50 subhalo 488530. Each frame = LEFT 4-quadrant off-axis projection
(Σ_gas, T, rest-frame v_LOS, N_HI) + photometric g/r/i stellar centre inset + bold sightline;
RIGHT a vertical stack of 10 ion absorption spectra (raw flux vs velocity, low→high IP).

Three continuous segments sweep impact parameter ρ, inclination, and azimuth α.
